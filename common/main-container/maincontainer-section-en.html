Section

This represents a section enclosed within the main container.

Code


    <h2>Main Container with "article"</h2>
    <main property="mainContentPage">
        <section>
        <!-- heading level 1 -->
    
        <!-- content page -->
    
        <!-- Page details section [Must be configured as a <footer> element in this situation] -->
    
        </section>
    </main>
